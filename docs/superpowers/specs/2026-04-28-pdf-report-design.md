# PDF Report Generation — Design Spec
**Date:** 2026-04-28
**Status:** Approved

---

## Overview

Add a downloadable PDF report to the SPC Control Chart Tool Streamlit app. The PDF is intended as a printable quality report — formatted and professional-looking — suitable for handing to a supervisor or including in process documentation.

The existing Excel download remains unchanged. The PDF is a separate, complementary output.

---

## Content & Layout

Single-page portrait PDF generated with `fpdf2`.

### 1. Header Block
- Title: "SPC Control Chart Report"
- Column name analyzed
- Chart type (X-bar/R, X-bar/S, or I-MR)
- Date generated (auto, from `datetime.date.today()`)

### 2. Chart
- The same matplotlib figure already rendered in the app (both subplots: X-bar and R/S/MR)
- Saved to a PNG buffer at 150 dpi, embedded full-width on the page
- Scaled to fit page width with aspect ratio preserved

### 3. Stats Table — 3-column grid
Three sections displayed side by side:

| Capability | Process | Control Limits |
|---|---|---|
| Cp | Mean (X̄) | UCL (X-bar) |
| Cpk | Sigma ST | CL (X-bar) |
| Pp | Sigma LT | LCL (X-bar) |
| Ppk | Min | USL |
| — | Max | LSL |
| — | Observations (N) | — |
| — | Subgroup Size (n) | — |

- Column headers bold
- N/A displayed for any None values
- Values rounded to 4 decimal places

### 4. Footer
"Generated by SPC Control Chart Tool — Internal Use Only"
Right-aligned, small font, at bottom of page.

## Implementation

### Dependency
- Add `fpdf2` to `requirements.txt`

### New function in `app.py`
`build_pdf(fig, stats, cap, column, chart_type, subgroup_size, USL, LSL) -> io.BytesIO`

Mirrors the signature of the existing `build_excel()` function for consistency.

Steps inside `build_pdf()`:
1. Save `fig` to a temporary PNG buffer (same as Excel export already does)
2. Write PNG buffer to a named temp file (fpdf2 requires a file path for images)
3. Create `FPDF` object, add page, set margins
4. Write header block
5. Embed chart image scaled to page width
6. Write 3-column stats table
7. Write footer
8. Return PDF as `io.BytesIO`

### UI change in `app.py`
Replace the single download button with two side-by-side buttons using `st.columns(2)`:
- Column 1: existing "Download Excel Report" button (unchanged)
- Column 2: new "Download PDF Report" button (`.pdf`, `application/pdf`)

Both buttons wrapped in the same `try/except` error handling pattern already used for Excel.

---

## Error Handling
- Wrap `build_pdf()` call in `try/except`, display `st.warning()` if it fails — same pattern as Excel.
- If the temp image file cannot be written, PDF generation fails gracefully without crashing the app.
